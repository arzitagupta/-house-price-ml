## House Price Prediction Model
This project is a Machine Learning web application that predicts house prices based on different features like area, bedrooms, bathrooms, and age.
## Technologies Used
- Python
- Scikit-learn
- Streamlit
## Features
- Predict house prices instantly
- User-friendly interface
- Trained ML model

## Project Structure
- app.py → main application
- train_model.py → model training
- dataset.csv → dataset
- house_model.pkl → trained model
- 
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py
